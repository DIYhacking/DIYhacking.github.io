Virtual Box and Ubuntu

Setting up Virtual box for running Ubuntu from my Mac OS.

Today we started off by installing virtual box for mac, this was done mainly to run Ubuntu from my mac system. This would make it easier to run fab modules and other programs that was made for Ubuntu.

I downloaded virtual box from here: https://www.virtualbox.org/ and installed it by following the instructions from here: http://osxdaily.com/2012/03/27/install-run-ubuntu-linux-virtualbox/

After this I was able to create a virtual system running on Ubuntu from my mac system.
